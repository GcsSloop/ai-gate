# Account Drag Overlay Minimal Design

## Goal
让账户卡片在拖拽排序时保持与当前极简页面风格一致，移除倾斜和缩宽等夸张浮层效果。

## Decision
采用极简方案：
- 保留拖拽浮层阴影，用于提示当前元素正在被拖拽。
- 保留原有占位卡片的高亮与虚线边框，用于提示落点。
- 移除拖拽浮层的旋转效果。
- 移除拖拽浮层的最大宽度限制，使其宽度跟随原卡片容器。

## Rationale
当前页面整体视觉语言是克制、稳定、规则的卡片布局。拖拽浮层的旋转和缩宽会引入更强的动效表达，与现有页面不一致，也容易让用户误判为尺寸异常。保留阴影和占位反馈即可满足交互提示，不需要额外戏剧化处理。

## Scope
仅调整前端账户列表拖拽浮层样式，并补充一个样式回归测试。
